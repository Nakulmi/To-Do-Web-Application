
# 📝 To-Do App

The To-Do App is a simple yet powerful web application that helps you organize and manage your daily tasks efficiently. You can add, mark as complete, and delete tasks — all with real-time updates and local storage support so your data is saved even after refreshing the page.


## ✨ Features

- ➕ Add Tasks: Easily add tasks with a single click or by pressing Enter.

- ✅ Mark Complete: Toggle tasks as complete/incomplete.

- ❌ Delete Tasks: Remove tasks you no longer need.

- 💾 Local Storage Support: Tasks are saved in your browser and persist after refresh.

- 📱 Responsive UI: Works smoothly on desktop and mobile browsers.


## 🛠️ Tech Stack

**HTML5** – For structuring the app

**CSS3** – For styling and responsive design

**JavaScript (ES6)** – For interactivity and logic

**LocalStorage API** – For storing tasks in the browser


## ⚙️ Installation

Clone this repository
```bash
git clone URL
```
Navigate to the project folder
```bash
cd todo-app
```
Open index.html in your browser

- You can double-click the file

- Or use a local server like Live Server in VS Code
## 🚀 Usage/Examples

- Type a task into the input box.

- Click "Add Task" or press Enter.

- Click on a task to mark it as completed.

- Click the "Delete" button to remove a task.

- Your tasks will automatically be saved and restored whenever you revisit the page.
